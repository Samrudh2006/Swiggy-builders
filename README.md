## 🚀 Quick Start

### 📋 Prerequisites

| Tool     | Min Version |
| -------- | ----------- |
| Node.js  | 20+         |
| Python   | 3.11+       |
| Docker   | 24+         |
| pnpm/bun | latest      |

### ⚡ One-Liner Setup

```bash
git clone https://github.com/Samrudh2006/swiggy-nexus-ai.git && \
cd swiggy-nexus-ai && \
cp .env.example .env && \
docker compose up -d && \
(cd backend && pip install -r requirements.txt && alembic upgrade head && uvicorn app.main:app --reload &) && \
(cd frontend && pnpm install && pnpm dev)
```

Open **http://localhost:3000** 🎉

### 🔑 Environment Variables

```env
🤖 AI
OPENAI_API_KEY=sk-...

🍽️ Swiggy MCP
SWIGGY_CLIENT_ID=your_client_id
SWIGGY_CLIENT_SECRET=your_client_secret
SWIGGY_MCP_ENDPOINT=https://mcp.swiggy.internal

🐘 Database
DATABASE_URL=postgresql://nexus:nexus@localhost:5432/nexus
REDIS_URL=redis://localhost:6379

🔐 Auth
JWT_SECRET=change-me-in-prod
```

## 🔌 API Reference

### 💬 Chat Endpoint

```http
POST /api/chat
Content-Type: application/json
Authorization: Bearer <token>

{
  "session_id": "sess_9f2a...",
  "message": "Something spicy under ₹300, please"
}
```

**Streaming Response** (SSE)

```json
{"type":"reasoning","text":"Searching spicy options under ₹300…"}
{"type":"tool_call","tool":"food.search","args":{"cuisine":"spicy","max_price":300}}
{"type":"tool_result","result":{"restaurants":[…]}}
{"type":"message","text":"Found 3 great picks. Want me to place the order?"}
{"type":"confirm_required","action":"place_order","cart_id":"cart_123"}
```

### 🛒 Order Confirmation

```http
POST /api/orders/confirm
{
  "cart_id": "cart_123",
  "payment_method": "upi"
}
```

### 🧬 Personalization

```http
GET  /api/preferences
PUT  /api/preferences
POST /api/preferences/forget
```

## 🔐 Security

- 🔒 **OAuth 2.1 + PKCE** for Swiggy authentication
- 🍪 **JWT rotation** in httpOnly, SameSite cookies
- 🚧 **Rate limiting** at gateway + agent boundaries
- 🛡️ **Prompt-injection guardrails** on all user inputs
- ✅ **Confirm-before-execute** on every destructive action
- 🔐 **Zero secrets in client bundle** — server-side only
- 🧾 **Full audit log** of every tool call
